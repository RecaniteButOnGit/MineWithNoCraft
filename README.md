# MineWithNoCraft (Minecraft Datapack)

Mine with no Craft (MWNC) is a custom survival datapack for **Minecraft Java 1.21.5** centered around:
- selling blocks/items at a beacon,
- buying tools and weapons with money,
- surviving without normal crafting progression.

## Core Rules

- Crafting is effectively disabled
- World is forced to daytime 
- Beacon location is randomized near spawn

## Installation

1. Go to Releases and copy the downloaded zip into your world's `datapacks/` directory
2. Rejoin the world or run `/reload` in-game.

Pack metadata:
- `pack_format: 71` (Minecraft Java 1.21.5)
